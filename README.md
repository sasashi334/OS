# OS

## 章ごとの感想
### 1章
1章では環境構築とバイナリエディタでのブートファイル作成, そしてC言語でのUEFIでのハローワールドを行った.
想定環境と私の環境のUbuntuのバージョンが異なっていたため, 想定バージョンのUbuntuを入れなおしたが, 少し苦労した.
バイナリエディタでの"Hello World!"は新鮮だった.

### 2章
2章ではブートローダ内でメモリマップの取得処理を行うプログラムの作成を行った.
C言語を触るのが久しぶりで, アドレスやポインタの良い復習になった. また関数ポインタという概念を知った. C言語の文法の固めてから, コードに対する理解力が格段に上がった気がする.
メモリマップをcsvファイルに出力できた時OSを作っている実感が湧いてとても良かった.

### 3章
