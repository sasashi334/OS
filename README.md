# OS

## 章ごとの感想
### 1章
1章では環境構築とバイナリエディタでのブートファイル作成, そしてC言語でのUEFIでのハローワールドを行った.
想定環境と私の環境のUbuntuのバージョンが異なっていたため, 想定バージョンのUbuntuを入れなおしたが, 少し苦労した.
バイナリエディタでの"Hello World!"は新鮮だった.

### 2章
2章ではブートローダ内でメモリマップの取得処理を行うプログラムの作成を行った.
C言語を触るのが久しぶりで, アドレスやポインタの良い復習になった. また関数ポインタという概念を知った. C言語の文法の固めてから, コードに対する理解力が格段に上がった気がする.
メモリマップをcsvファイルに出力できた時OSを作っている実感が湧いてとても良かった.

### 3章
3章ではカーネルの作成と画面描画をおこなった。
UEFIのGOPからフレームバッファを取得して描画する。
### 4章
makefileや画面描画を行う。
今まで参照渡しをポインタ渡しと勘違いしていたことが判明。参照渡しをとても便利に感じたためこれから使っていきたい。
またここで初めてクラスが出てくる。自作OSではメモリ管理を自分で行うため通常のnewではなく配置newというものを用いることを学んだ。

### 5章
コンソールを作成した。文字列表示の仕組みが知れてよかった。
### 6章
マウス入力の実装。
USBドライバを使って実装したが、とても複雑であった。

### 7章
前半ではマウス操作をポーリング方式から割り込み方式に変更。後半はqueueの実装。
この章では最適化の側面が強く見栄えは変わらなかったがqueueの実装はとても興味深かった。

### 8章
メモリ管理の基礎を作成した。
普段裏側で動いているメモリ管理の仕組みの一端が知れた。
また、ユーザー定義リテラルという機能を学んだ。かなり可読性が上がるので積極的に使っていきたい。
